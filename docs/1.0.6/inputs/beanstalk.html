beanstalk

Pull events from a beanstalk tube.

TODO(sissel): Document where to learn more about beanstalk.

Synopsis

This is what it might look like in your config file: 

input {
  beanstalk {
    debug => ... # boolean
    host => ... # string (required)
    port => ... # number, default: 11300
    tags => ... # array
    tube => ... # string (required)
    type => ... # string (required)
  }
}

Details

debug

  • Value type is boolean
  • There is no default value for this setting.

Set this to true to enable debugging on an input.

host (required setting)

  • Value type is string
  • There is no default value for this setting.

The address of the beanstalk server

port

  • Value type is number
  • Default value is 11300

The port of your beanstalk server

tags

  • Value type is array
  • There is no default value for this setting.

Add any number of arbitrary tags to your event.

This can help with processing later.

tube (required setting)

  • Value type is string
  • There is no default value for this setting.

The name of the beanstalk tube

type (required setting)

  • Value type is string
  • There is no default value for this setting.

Label this input with a type.
